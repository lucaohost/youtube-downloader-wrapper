**Instalation**
---
- Install npm if you didn't have yet (https://www.npmjs.com/get-npm)
- In the terminal, run:
    - git clone git@github.com:lucaohost/video-youtube-downloader.git
    - cd video-youtube-downloader.git
    - npm install
**Usage**
---
- Put a youtube URL in video-download.js line 1 or playlist-download.js
- In the project, open terminal
- Run in the terminal node video-download.js or playlist-download.js
- The videos donwloaded are in the folder of project
